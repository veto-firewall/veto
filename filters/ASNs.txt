# ASN (Autonomous System Number) Filter Examples

# Format:
# - Enter ASN numbers separated by spaces or new lines
# - ASNs are used by large organizations for routing traffic on the internet
# - Each number represents a network owned by a specific organization
#
# How to find ASN numbers:
# 1. Visit https://ipinfo.io/asn-lookup or https://bgp.tools/
# 2. Search for a company name (e.g., "Microsoft", "Facebook")
# 3. Look for the ASN number (usually shown as "ASXXXX")

# - https://ipinfo.io/asn-lookup
